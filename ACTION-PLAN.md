# SERP Strategists SEO Action Plan

## Critical

- None. The site is indexable and the live homepage is rendering correctly.

## High

1. Rewrite `llms.txt` so the resource list uses markdown link syntax.
2. Add `Organization` schema to the homepage and `Article` / `BlogPosting` schema to posts.

## Medium

1. Fix the low-contrast homepage microcopy and queue labels.
2. Correct the heading hierarchy so heading levels descend sequentially.
3. Add `BreadcrumbList` schema across the core page templates.
4. Reduce homepage DOM size and review whether the marquee / duplicated logo blocks need to render as many nodes.
5. Audit Google Tag Manager usage and defer any non-essential third-party work.
6. Replace deprecated sitemap `changefreq` and `priority` tags with a cleaner sitemap.

## Low

1. Add richer credibility signals on the about and governance pages.
2. Tighten utility-page copy where it helps, especially FAQ and contact.
3. Consider trimming the amount of repeated hero whitespace on long scroll pages.

## Suggested Order

1. Fix `llms.txt`.
2. Fix contrast and heading order.
3. Ship schema improvements.
4. Trim DOM / third-party cost.
5. Clean up sitemap formatting.
